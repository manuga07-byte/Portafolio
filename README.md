
# Portfolio Profesional Bootstrap 5

## Descripción

Portfolio personal desarrollado con Bootstrap 5 para mostrar mis proyectos, habilidades técnicas y experiencia como desarrollador web.

## Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## Características

* Diseño responsive
* Navbar adaptable
* Cards para proyectos
* Modal informativo
* Carousel de imágenes
* Formulario de contacto

## Instalación

1. Clonar repositorio
2. Abrir `index.html` en el navegador

## Despliegue

Aplicación publicada mediante GitHub Pages.
